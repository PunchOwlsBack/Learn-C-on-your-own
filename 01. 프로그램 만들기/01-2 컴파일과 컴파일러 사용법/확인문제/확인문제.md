### 3번
자신의 학번, 이름, 학점을 출력하는 프로그램이 완성되도록 빈칸에 알맞은 코드를 적으세요.
- 학번은 정수, 이름은 문자열, 학점은 문자 상수를 사용합니다.
  
```C
#include <stdio.h>

int main(void)
{
    printf("학번 : %d\n", 32165);
    printf("이름 : %s\n", "홍길동");
    printf("학점 : %c\n", 'C');
}
```
